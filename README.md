Travel Agency System

Description This project is a travel agency platform that allows users to book transportation services such as cars and vans, manage reservations, and rate their experience. The system includes both a client interface and an admin dashboard for managing operations.

Tech Stack

Backend: (Node.js)
Frontend: (React js)
Database: (MySQL)
CI/CD: Jenkins
Key Features

Vehicle reservation system (cars, vans, transportation services)
Simulated payment integration (PayPal & Google Pay - Sandbox)
Admin dashboard for managing trips and bookings
Customer interface for reservations and service interaction
Notification system for booking updates
Rating and review system after service completion
CI/CD Automation

Implemented CI/CD pipelines using Jenkins
Automated execution of frontend and backend tests on each commit to the dev branch
Automatic merge to main branch after successful validation
Ensured continuous integration and reliable deployments
Architecture Designed as a full stack web application with a focus on modularity and maintainability, applying clean architecture principles.

How to Run

Clone the repository
Install dependencies
Configure environment variables
Run the application
